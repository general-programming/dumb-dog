## Host!

You can host a Dumb Dog instance yourself, if you wish.

```sh
git clone https://github.com/general-programming/dumb-dog.git
cd dumb-dog
gradle shadowJar
java -jar build/libs/dumbdog-2.0-SNAPSHOT-all.jar
```
